<h3>◆概要</h3>
<li>Blue Protocol: Star Resonanceの蝕花の残影におけるギミックの一つ簡易コードブレイカーの補助を目的としたもの。</li>
<li>現状でこれ以上の機能を追加する予定はありませんができうる限りの改善はいたします。</li>
<h3>◆説明</h3>
<li>通常モードでは入力コードと正解数から候補を表示します。</li>
<li>指示モードでは指示されたコードに対する正解数を入力することで更に次のコードを指示し確定に導きます。</li>
